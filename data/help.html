Notes Scoreboard Help..v0.1 The scores and games are set using the up and down buttons. The player at each end can be selected. e.g. Home1, Away1 etc. The Menu provides the following functions:- Refresh - refreshes the scores. Scorecard - displays the current match scores. Download - the scorelist. Each game in sequence of being saved Manage creation of a scorecard. Another menu is opened where a scorecard like file can be produced Save - The currently displayed score can be saved to the scoresheet. Service Won indicates who wins the toss and can be used as a reminder for which end one the toss. Change ends will swap over score and players. Can also be used in combinaton with clear points. Clear points clears the current points score. Clear points and games clears all points. Clear match clears all points, games and deletes the scorecard.
